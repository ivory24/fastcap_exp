第一章核心思想：
1.1
1.实际问题转化为数学问题，数学问题转化为程序问题。
2.>> r = 6367   意思是：一个变量名是r的盒子里面存了一个值6367
注意：并不是一个代数等式，而是把右边的值赋给左边
3.变量名必须以字母开始，由字母、数字、还有下划线组成。变量名在使用之前应该先定义。
4.我们可以直接写一个表达式里面有很多的运算，不需要把每一步的值都存下来，matlab在幕后会做这件事情【存每一步的值】。
5.显示结果时：一行的结尾不用打分号【；】。
比如：
>> r = 6367
r =
	6367
6.一个数的值比较大时，它会用科学计数法输出。一般会精确到小数点后四位。
7.format可以改变matlab输出的数【比如精确度更高】：
>> format long
>> A = 4*3.1416*.6367^2
A =
	5.094253814496001
还有
format short    小数位，5位
format long     小数位，15位
format short e  科学计数法，5位小数
format long e   科学技术法，15位小数
8.注释%开头，给每一个脚本命名，解释脚本的作用。当程序比较复杂时，我们需要借助注释来澄清一段一段代码和变量的作用。
……的作用是：一行的表达式没有写完，和下一行要连接起来。
clc:清除窗口，输出答案。
disp（’string‘）：被用来输出字符串
fprintf（‘%10.6f’）：10个字符，三个整数部分，一个字符被用作小数点，小数部分精确到小数点后六位
input:里面的字符串提醒matlab用户输入数据，脚本会在数据输入之后运行。
output:有特定的字母表示输出的格式：分配10个空间用来显示数，分配三个空间来显示小数点的右边。？
f:标准的小数记号
e:科学计数法
\n:换行符
m-file必须有一个名字：比如SurfArea.   作用：存储脚本
一旦建立一个m-file，我们就可以运行这个脚本通过输进m-file的名字，脚本应放在当前目录【会显示在matlab输出框的一个文本输入框里面】下运行。
9.计算机的不精确：同一个表达式的计算顺序不同，结果也会有差异
10.字符串是由单引号引起来的，单引号中间也可以有单引号。
11.一行可以有多个表达式，但是需要用分号或者逗号隔开。
12.数学公式：
负数开根号，会出现一个神奇的值；so负数不能开根号    sqrt（4）
